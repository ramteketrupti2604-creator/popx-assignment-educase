
# PopX - Authentication Flow UI

This project is a pixel-perfect implementation of the PopX mobile app authentication flow as part of the Educase India React JS internship assignment.

 
## ✨ Features

- **Pixel-perfect UI** matching the provided Adobe XD design
- **4 Complete Screens**: Landing, Login, Signup, and Profile
- **Fully Responsive** mobile-first design (375px x 620px frame)
- **Interactive Navigation** between all screens
- **Form Validation Ready** UI with proper input fields
- **Clean React Structure** with component-based architecture

## 🛠️ Tech Stack

- **React 18** - UI Library
- **Vite** - Build tool & dev server
- **React Router DOM** - Client-side routing
- **CSS3** - Custom styling without external UI libraries
- **Rubik Font** - As per design specifications

## 💻 How to Run Locally

1. **Clone the repository**
   https://github.com/ramteketrupti2604-creator/popx-assignment-educase


2. *Install dependencies*
   npm install


3. *Start the development server*
   npm run dev


4. *Open in browser*
   http://localhost:5174
